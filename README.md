**Где:**
- **40%** - генетическое влияние родителей (`w_g = 0.4`)
- **60%** - влияние окружающей среды (`w_e = 0.6`)
- **G_i** - генетическая предрасположенность
- **E_i** - факторы среды (школа, друзья, семья)
